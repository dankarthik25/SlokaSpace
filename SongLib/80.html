Sloka No:     1  

Text | 1 |

ālāyāl taṟa vēṇaṁ aṭuttorampalaṁ vēṇaṁ
ālinu cērnnoru kuḷavuṁ vēṇaṁ
kuḷippānāy kuḷaṁ vēṇaṁkuḷattil centāmara vēṇaṁ
kuḷicc cennakaṁ pukkān candanaṁ vēṇaṁ

Synonyms

āl(ആല്‍):Banyan Tree ; āyāl:If possible ; taṟa:raised floor ; vēṇaṁ:need ; aṭuthoru:next to , near by ; āmbalam:Temlpe ; vēṇaṁ:need ;
ālinu:Banyan Tree ; cērnnoru:near by ; kuḷavuṁ:Lake/Pond ; vēṇaṁ:need ;
kuḷippānāy:To take a bath ; kuḷaṁ:Lake/pond ; vēṇaṁ:need ; kuḷattil:In the Lake/pond ; centāmara:Louts Flower ; vēṇaṁ:need ;
kuḷicc:Taken a bath ; cennakaṁ:cennakaṁ ; pukkān:fragrance,smoke ; candanaṁ:sandalwood ; vēṇaṁ:need ;

Translation

If there‘s banyan tree, it needs to have a raised floor, and a temple next to it
There is also needs to be a lake close to the banyan tree
There needs to be a lake to take bath
There need to be sandal wood paste to be used as perfume after the bath
