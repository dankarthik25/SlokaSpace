Sloka No:     1   2   3  

Text | 1 |

tum meri rākho lāja hari
tum jānata saba antarayāmī karanī kachu na karī

Synonyms

tum:You ; meri:my ; rākho:keep ; lāja:modesty/honor ; hari:Hari ; tum:You ; jānata:know ; saba:everything ; antarayāmī:Omniscient One ; karanī:doer ; kachu:something ; na:not ;

Translation

Please protect my honor, O Hari. You know everything, O Omniscient One. The doer does not do anything.

Text | 2 |

avaguṇa mose bisarata nāhi pala china gharī gharī
sab prapañca kī poṭa bāńdh kari apne sīs dharī

Synonyms

avaguṇa:faults ; mose:me ; bisarata:forget ; nāhi:not ; pala:moment ; china:lost ; gharī:moment ; gharī:moment ; sab:all ; prapañca:wordly ; kī:of ; poṭa:bundle ; bāńdh:tied ; kari:do ; apne:my own ; sīs:head ;

Translation

2) I am unable to be redeemed of my faults as each moment passes. I carry a bundle bound of this worldly existence upon my head.

Text | 3 |

dārā suta dhana moha liye hauń sudhi-budhi saba bisarī
sūra patita ko bega udhāro ab merī nāva bharī

Synonyms

dārā:friends ; suta:sons ; dhana:wealth ; moha:attachments ; liye:took away ; hauń:You have ; sudhi:body ; budhi:mind ; saba:all ; bisarī:left me ; sūra:Sura Dasa ; patita:the fallen ; ko:of ; bega:quickly ; udhāro:please deliver ; ab:now ; merī:my ; nāva:boat ; bharī:filled ;

Translation

3) My friends, sons, wealth, attachments, and everything I own have left me. The fallen and destitute, Sura Dasa, begs to be delivered. Now my boat is filled (i.e. my life will thus be completed).
